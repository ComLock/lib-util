# lib-util

## Usage

### Gradle build script

    repositories {
        maven {
            url 'http://repo.enonic.net/public'
        }
    }

    dependencies {
        include 'com.enonic.lib:util:1.0.0'
    }

### Controllers

    var UTIL = require('/lib/enonic/util/util');
